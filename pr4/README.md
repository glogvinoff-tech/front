За шесть практических работ в рамках курса «Фронтенд и бэкенд разработка» был пройден полный стек создания веб-приложения: начиная с изучения CSS-препроцессоров (SASS/LESS) для модульной и поддерживаемой вёрстки, далее — настройка сервера на Node.js + Express с реализацией CRUD-операций, работа с форматом JSON и тестирование API через Postman, интеграция бэкенда с React-фронтендом с использованием axios и решение CORS-проблем, расширение API документацией через Swagger (JSDoc + swagger-ui-express), и завершающий этап — подготовка к контрольной работе: финальное тестирование, оформление README и публикация репозитория. В результате сформирован рабочий прототип fullstack-приложения (например, каталог пользователей или товаров) с чёткой архитектурой, документацией и готовностью к дальнейшему масштабированию.
